## Git Code
    1. git add file.name
        a. 자신이 바꾼 파일 전부다 add 해야 함.
        b. 이거 할때는 같은 directory에 있어야 합니당.
    2. git commit -m "쓰고싶은거 아무거나"
        a. 자신이 바꾼 이유, 아주 간략한 메세지
    3. git push origin main
        a. 위에 2개 다하고 github에 업로드 할때 쓰는 커맨드
    4. git pull origin
